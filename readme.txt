Automated DLE To WordPress Migration Plugin

== Description ==

You have a lot of content on your DataLife Engine website. But you’ve considered that WordPress CMS suits you better. You start thinking of moving DataLife Engine website content to WordPress. You’re mostly interested in a flawless, accurate and effortless migration, since you may be a busy person or with little coding skills.

= How does this plugin help: =
- Free Demo Migration allows you to see the results of the migration beforehand;
- 15 min is the average migration time (Note: timing depends on how many content items you’re going to move from DataLife Engine to WordPress);
- Migrate the content you want. Choose how you want it to be and look on WordPress (Data ‘Mapping’ option);
- Complete data migration price depends on the content volume you’re about to migrate to WordPress.

= What you can move from DataLife Engine to WordPress: =
* Pages along with the content included there;
* Images included in the content body.

= Preserved SEO rankings after the migration =
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing DLE website to the new WordPress website.

= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Live help is available. Our support team is here 24/7.
- When a custom migration is needed, CMS2CMS offers data migration service packages to order.

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated DLE To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
